# 디스코드 채팅 내역 생성기

스트리머의 대화 내용을 입력하거나 CSV로 불러오면 디스코드 채팅 내역처럼 렌더링하고 PNG/WebM으로 저장하는 웹앱입니다.

## 실행

```bash
https://chat-log-studio.vercel.app
```

브라우저에서 `(https://chat-log-studio.vercel.app/)`을 엽니다.

## 대화 입력 형식

```txt
A: 야 디코 좀 봐
B: 나 오늘 로아 접속 안 됨
```

프로필 영역에서 표시 이름과 프로필 사진을 설정할 수 있습니다. 프로필 사진 원형 버튼을 누르면 이미지를 업로드합니다.

CSV를 불러오면 이름 열, 메시지 열, 시간 열을 선택해 대화 입력란으로 변환할 수 있습니다. WebM 저장은 메시지가 순서대로 나타나는 영상으로 내보냅니다.
